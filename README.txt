මෙම software එක භාවිතා කරන විදිහ:

1. Python install කරගන්න
2. Terminal එකෙන් `pip install openpyxl` කියලා run කරන්න
3. `main.py` run කරන්න → GUI එක open වෙනවා
4. Item එක select කරලා Qty enter කරන්න
5. Save Order click කරන්න
6. Orders Excel sheet එකට save වෙනවා
7. `income_checker.py` run කරලා අද ආදායම බලන්න
